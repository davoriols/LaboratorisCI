# Computer Interfacing Laboratory exercises

## Overview
This repository contains the different laboratory practices of the Computer Interfacing course at FIB-UPC, done in collaboration in teams of two.
Each practice of laboratory may be divided in more than one session, each one containing a previous work (required to successfully complete the lab exercise) and the lab exercise itself.

## Usage
The 8.13 version of Proteus is required to run the schematics, as the Software has no retrocompatibility. More dependencies may be required as sessions increase in complexity.
Further notice will be updated here.
